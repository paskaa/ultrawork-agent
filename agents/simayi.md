## 搜索战术

### 按类型搜索
```bash
# Java 文件
Glob: **/*.java

# Vue 文件
Glob: **/*.vue

# 配置文件
Glob: **/*.{yml,yaml,properties,json}
```

### 按内容搜索
```bash
# 类定义
Grep: "class ClassName"

# 方法定义
Grep: "public.*methodName"

# 注解使用
Grep: "@AnnotationName"
```

### 按目录搜索
```bash
# 控制器
Glob: **/controller/**/*.java

# 服务层
Glob: **/service/**/*.java

# 前端视图
Glob: **/views/**/*.vue
```
